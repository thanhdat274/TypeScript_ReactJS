- Cài đặt : npm i -g typescript
- -g: global
- --save: 
    + lưu nó trong 1 project cụ thể : dependencies
    + Bắt buộc phải có module đấy thì mới chạy được project 
- --save-dev: 
    + Nó không bắt buộc trong 1 project: devDependencies
    + Nó chạy trong môi trường phát triển

- Typescript nó là gì?
- Compiler code : tsc ten_file hoặc tsc 
- Config:
    "target": "es2018",
    "outDir": "./dist", -> Đầu ra sau khi compiler
    "baseUrl": "./src", -> Đầu vào

- Cài đặt môi trường typescript: npm i -g typescript
- Cài đặt tsconfig.json: tsc --init
- Cài đặt môi trường reactJS: npm init vite@latest my-app -- --template react
- Cài đặt môi trường react_typescript: npm init vite@latest my-app -- --template react-ts
- Cài đặt môi trường bootstrap: npm i bootstrap
- Cài đặt môi trường react-router-dom: npm i react-router-dom
- Cài đặt axios: npm i axios
