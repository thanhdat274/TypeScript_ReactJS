- Cài đặt : npm i -g typescript
- -g: global
- --save: 
    + lưu nó trong 1 project cụ thể : dependencies
    + Bắt buộc phải có module đấy thì mới chạy được project 
- --save-dev: 
    + Nó không bắt buộc trong 1 project: devDependencies
    + Nó chạy trong môi trường phát triển

- Typescript nó là gì?
- Compiler code : tsc ten_file hoặc tsc 
- Config:
    "target": "es2018",
    "outDir": "./dist", -> Đầu ra sau khi compiler
    "baseUrl": "./src", -> Đầu vào
